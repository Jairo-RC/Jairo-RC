## 🔬 Home SOC Lab Environment

I have engineered and deployed a fully functional, highly segmented **Home Security Operations Center (SOC)** to simulate real-world enterprise architectures and attacks. This practical lab enables me to test defensive capabilities against active threats.

- **Core Infrastructure:** Deployed **Wazuh SIEM**, **Suricata IDS**, **Snort IPS**, and a **pfSense Firewall** for comprehensive network boundary and endpoint protection.
- **Topology & Segmentation:** Designed with isolated zones including a secure LAN, a DMZ for exposed services, and a dedicated Management interface for administrative monitoring.
- **Endpoint Monitoring:** Actively monitoring **8+ endpoints** (Linux and Windows) with advanced logging and telemetry collection.
- **Threat Detection & Correlation:** Authored custom correlation rules inside Wazuh to detect specific TTPs such as port scans, brute-force attempts, and indicators of lateral movement.
- **Traffic Analysis & Vulnerability Management:** Conducting continuous network traffic analysis and scheduled vulnerability assessments using **OpenVAS** and **Nmap** to maintain a hardened attack surface.
